# Download endpoint

The path to download the files, using the REST api, will be:

* host://
* website.it
* :port
* /snello-api/documents/
* {uuid}
* /download

```text
http://localhost/snello-api/api/documents/f6978b5a-e2ba-4991-8c66-0e0f803b9f87/download
```
