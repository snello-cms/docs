# Metadata

The **metadata** is the join between database and api.  Snello can:

* create a new table for you
* use an existent table in the database.

**The required field:**

* label - table name
* description
* icon

**It's a new Table? YES**

* Table key
* Table key type \(uuid, slug, auto increment\)

**It's a new Table? NO**

* Select Fields 
* Search Fields 
* Alias table 
* Alias condition
*  Table key
* _Table key Type_

**Advanced Options?**

* Creation query
* Tab & Groups
* Order By
