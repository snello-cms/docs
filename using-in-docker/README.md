# Docker Images

## From docker hub

You can find all SNELLO images in the docker hub

```
https://hub.docker.com/u/snellocms
```

In three flavors:

{% hint style="info" %}
 Snello API
{% endhint %}

```text
docker pull snellocms/snello-api
```

{% hint style="success" %}
Snello Admin
{% endhint %}

```text
docker pull snellocms/snello-admin
```

{% hint style="danger" %}
Snello All in One
{% endhint %}

```text
 docker pull snellocms/snello-all-in-one
```
