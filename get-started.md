# Get Started

## \[1] install docker

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/snellocms/install/HEAD/install.sh)"
```

## \[2] download & run snello-all-in-one image

```
docker compose up
```

## \[3] open the browser

```
http://localhost:8080/snello-admin
```
